```
usage: matrix.py [-h] [--max_active_lines MAX_ACTIVE_LINES] [--freedom] [--speed SPEED] [--group GROUP] file_name

Matrix-style scrolling text effect.

positional arguments:
  file_name             Path to the file containing text.

options:
  -h, --help            show this help message and exit
  --max_active_lines MAX_ACTIVE_LINES
                        Maximum number of lines visible at a time.
  --freedom             Make text fall from the top.
  --speed SPEED         Set the scroll speed (higher is faster).
  --group GROUP         Set the grouping number.
```
